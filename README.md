# κατεδαφίσεις

Εργαλείο έρευνας για τη χαρτογράφηση των αδειών κατεδάφισης στην Ελλάδα
(στο πνεύμα του [«Ο διαρκής θάνατος της μονοκατοικίας»](https://www.kathimerini.gr/investigations/563435530/o-diarkis-thanatos-tis-monokatoikias/)).

Αντλεί από τη **Διαύγεια** τις τελικές «Άδειες Κατεδάφισης» που δημοσιεύει το
ΤΕΕ μέσω του e-Άδειες, κατεβάζει το PDF κάθε άδειας, εξάγει τα στοιχεία
διεύθυνσης και το **περίγραμμα του κτίσματος** (συντεταγμένες ΕΓΣΑ87 από τη
φόρμα, μετατροπή σε WGS84 — η συντριπτική πλειονότητα των αδειών το έχει)
και παράγει spreadsheet. Όπου το PDF δεν δίνει συντεταγμένες, γεωκωδικοποιεί
τη διεύθυνση με Nominatim/OpenStreetMap.

## Εγκατάσταση

Χρειάζονται **Python 3** και το **pdftotext** (πακέτο poppler). Σε
Debian/Ubuntu/Mint όλα μπαίνουν με μία εντολή:

```sh
sudo apt install python3-requests python3-openpyxl python3-flask poppler-utils
```

Εναλλακτικά (macOS, ή όποιος προτιμά pip) με virtualenv — στα σύγχρονα
Linux το σκέτο `pip install` εκτός venv δεν επιτρέπεται:

```sh
brew install poppler                     # macOS· σε Linux: sudo apt install poppler-utils
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt    # requests, openpyxl, flask
```

(Με venv, στις παρακάτω εντολές αντικαταστήστε το `python3` με
`.venv/bin/python3`.)

## Εκτέλεση — Web UI (για μη ειδικούς)

```sh
python3 webui.py
```

Ανοίγει μόνο του τον browser σε μια απλή σελίδα: επιλογή περιοχής με
autocomplete, διάστημα ημερομηνιών, ζωντανή πρόοδος, και στο τέλος κουμπιά
λήψης (Excel, zip με όλα τα PDF) και διαδραστικός πίνακας με ταξινόμηση
και φίλτρα. Με το κουμπί «Χάρτης» τα αποτελέσματα εμφανίζονται ως σημεία
σε χάρτη OpenStreetMap (ομαδοποιημένα, χρωματισμένα κατά ακρίβεια
γεωκωδικοποίησης, με popup ανά άδεια)· τα φίλτρα του πίνακα ισχύουν και
στον χάρτη. Το spreadsheet βγαίνει **αμέσως**· οι συντεταγμένες
προστίθενται μετά ως εμπλουτισμός (μπορεί να ακυρωθεί και να συνεχιστεί
αργότερα με το κουμπί «Προσθήκη συντεταγμένων»). Η καρτέλα «Ιστορικό»
δείχνει όλα τα παλιότερα runs (αποθηκεύονται στο `runs/`).

Τερματισμός: Ctrl-C στο τερματικό. Η σελίδα τρέχει μόνο τοπικά
(127.0.0.1) — δεν είναι προσβάσιμη από το δίκτυο.

## Εκτέλεση — γραμμή εντολών

```sh
python3 katedafiseis.py --area "Δήμος Δράμας" --from 01/01/2021 --to 31/12/2021 -o drama
python3 katedafiseis.py --area "Νομός Καβάλας" --from 01/01/2019 -o kavala
python3 katedafiseis.py --area "Περιφέρεια Κρήτης" --from 01/01/2024 -o kriti
python3 katedafiseis.py --area "Ελλάδα" --from 01/01/2023 --to 31/12/2023 --no-geocode -o ellada2023
```

Ημερομηνίες σε μορφή ΗΗ/ΜΜ/ΕΕΕΕ (δεκτή και η ΕΕΕΕ-ΜΜ-ΗΗ).

Κάθε run παράγει έναν φάκελο:

```
drama/
├── drama.xlsx
└── pdf/<δήμος>/<έτος>/<ΑΔΑ>.pdf   # όλα τα PDF των αδειών του run
```

Η στήλη «PDF» του spreadsheet δείχνει στο τοπικό αρχείο (σχετική διαδρομή —
ο φάκελος μεταφέρεται/μοιράζεται αυτούσιος) και η στήλη «ΑΔΑ» στη Διαύγεια.

`--area` δέχεται δήμο, νομό/περιφερειακή ενότητα, περιφέρεια, «Ελλάδα», ή
πολλά χωρισμένα με κόμμα. Σκέτο όνομα νομού (π.χ. «Καβάλας», «Μαγνησίας»,
«Κυκλάδων») ερμηνεύεται ως ο παλιός νομός· για τη στενότερη περιφερειακή
ενότητα γράψτε «ΠΕ Καβάλας». `--no-geocode` παραλείπει τις συντεταγμένες
(πολύ πιο γρήγορο για μεγάλες περιοχές).

## Το spreadsheet

- **Κατεδαφίσεις** — μία γραμμή ανά άδεια: ΑΔΑ (link στη Διαύγεια), ημερομηνία,
  έτος, δήμος, δημοτική ενότητα/περιοχή, πόλη/οικισμός, οδός, αρ. από/έως, ΟΤ,
  ΚΑΕΚ, περιγραφή έργου, όροφοι (1 = ισόγεια/μονώροφη, 2 = διώροφη, …, κενό =
  δεν προσδιορίζεται), συντεταγμένες, ακρίβεια θέσης
  (κτίσμα (PDF) / οδός+αριθμός / οδός / οικισμός / δήμος).
- **Ανά έτος-δήμο** — pivot με πλήθη ανά έτος και δήμο.

Η στήλη «Όροφοι» επιτρέπει φιλτράρισμα σε μονώροφες/διώροφες (το ζητούμενο
της έρευνας) χωρίς να χάνονται οι υπόλοιπες εγγραφές.

Η στήλη «Έλεγχος» σημαδεύει δύο γνωστές παθολογίες των δεδομένων (επιβεβαιωμένες
σε σύγκριση με την έρευνα της Καθημερινής για την Αττική):

- **«πιθανό διπλό»** — το ίδιο κτίσμα παίρνει συχνά περισσότερες από μία
  τελικές άδειες (επανεκδόσεις με ίδια διεύθυνση/περιγραφή, διαφορετικό ΑΔΑ).
  Για μέτρηση κτισμάτων αγνοήστε τις γραμμές αυτές.
- **«~Xkm από τον δήμο»** — οι συντεταγμένες πέφτουν μακριά από τον δήμο των
  μεταδεδομένων· συνήθως λάθος επιλογή δήμου στο e-Άδειες (π.χ. άδειες του
  Ηρακλείου Κρήτης δηλωμένες στον ομώνυμο δήμο της Αττικής — μοναδικό
  διπλώνυμο ζευγάρι δήμων στην Ελλάδα).

## Τι μετράει

Μόνο τις **τελικές** «Άδειες Κατεδάφισης» (μία ανά κτίσμα). Δεν μετράει:
προεγκρίσεις, αναθεωρήσεις/ενημερώσεις (διπλομέτρημα του ίδιου κτίσματος),
ούτε κατεδαφίσεις ενσωματωμένες σε ενιαία «Οικοδομική Άδεια»
κατεδάφισης‑και‑ανέγερσης — ο πραγματικός αριθμός χαμένων κτισμάτων είναι
άρα κάπως μεγαλύτερος.

## Περιορισμοί

- Κάλυψη από **Οκτώβριο 2018** (έναρξη e-Άδειες)· παλαιότερες άδειες δεν
  υπάρχουν ομοιόμορφα στη Διαύγεια.
- Για τις λίγες άδειες χωρίς συντεταγμένες στο PDF, η γεωκωδικοποίηση
  εξαρτάται από το OSM· ακίνητα χωρίς οδό/αριθμό πέφτουν στο κέντρο του
  οικισμού ή του δήμου (βλ. στήλη «Ακρίβεια συντ/νων»).
- Το Nominatim έχει όριο ~1 αίτημα/δευτερόλεπτο· για μεγάλες περιοχές η
  πρώτη εκτέλεση αργεί (τα αποτελέσματα κρατιούνται σε cache).

## Cache

Όλα (αναζητήσεις, PDF, γεωκωδικοποιήσεις) αποθηκεύονται στο `cache/`· οι
επανεκτελέσεις είναι σχεδόν ακαριαίες. Σβήστε το `cache/search/` για να
ξαναφέρει πρόσφατα δημοσιευμένες πράξεις σε ήδη κατεβασμένα διαστήματα.
